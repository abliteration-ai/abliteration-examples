# Policy Gateway connectors

Small, open-source connectors that make Policy Gateway adoption a 5-minute task.

Included connectors:
- policy-gateway-nextjs: Next.js App Router route handler wrapper.
- policy-gateway-fastapi: FastAPI dependency + client helper.
- policy-gateway-langchain: LangChain wrapper + callback for audit metadata.
- policy-gateway-litellm: LiteLLM helper for routing through /policy.

Each connector includes:
- A 5-minute quickstart.
- A local-dev mode using a local Policy Gateway base URL.
- Sample log output for dashboards and audits.
